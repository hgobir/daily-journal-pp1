# daily-journal-pp1



## Part 1 - Markdown Journal Writer



Create a markdown journal system that:

- Date-based entries (one file per day: YYYY-MM-DD.md)
- Stores entries in journal/ folder
- Randomized reflection prompts
- Commands: new, list, search, open
- Markdown formatting for entries
- Timestamp each entry
- Clean CLI interface with argparse
- Search through past entries
- Open entries in your editor

Skills: file organization, markdown formatting, command-line tools, argparse, pathlib, datetime handling, and building daily-use tools
